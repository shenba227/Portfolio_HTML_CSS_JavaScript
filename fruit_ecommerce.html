Fruit Ecommerce Website

This project showcases an ecommerce website for fruits, developed using React. It features multiple components and navigation to provide a dynamic and interactive user experience. Users can explore various fruit products, view details, and add items to their cart.

Key Features:

  • Component-based architecture using React for modular and maintainable code.
  • Dynamic routing and navigation between product pages and shopping cart.
  • Interactive UI elements for adding items to the cart and viewing product details.
  • State management to handle user interactions and data.

Technologies Used:

  • React for building the user interface and managing application state.
  • HTML and CSS for overall structure and styling.
