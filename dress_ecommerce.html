Dress Ecommerce Website

This project is a simple ecommerce website designed to showcase and sell dresses. Built with HTML and CSS, it features a clean and user-friendly interface that allows users to browse through various dress collections and view product details.

Key Features:

  • Responsive layout that adapts to different screen sizes.
  • Product catalog with filter and search functionality.
  • Stylish product pages with detailed descriptions and images.
  • Basic cart functionality to simulate adding products.

Technologies Used:

  • HTML for structuring the website content.
  • CSS for styling and layout design.
